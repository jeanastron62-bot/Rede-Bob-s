# Rede-Bob-s
Rede Trailer Bob’s Burger é um projeto de automação criado para digitalizar o atendimento e a gestão de pedidos de um trailer de hambúrguer. Substitui comandas de papel por um sistema em celular ou tablet, trazendo mais agilidade, organização, redução de erros e controle da operação, com possibilidade de expansão futura.
